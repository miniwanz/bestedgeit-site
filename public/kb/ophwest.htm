Outlook Profile Helper Instructions and Features:

To make creating Outlook profiles easier and more streamlined, we have provided an Application called the Outlook Profile Helper (OPH). The OPH consists of the following features: 

  • Automatically creates an Outlook Profile with limited input
  • Allows you to create multiple profiles on the same PC
  • Allows you to save your password for Outlook when creating your profile

Using the Outlook Profile Helper:

Important: Outlook must be closed before running the Outlook profile Helper. This wizard will create a new profile and will not migrate existing mailboxes or .pst files (does not delete or remove files).

Step 1: Download and launch the Outlook Profile Helper file here à BE_OutlookW.exe

·         If a profile named ‘Exchange’ already exists, you will be asked if you want to overwrite the existing profile.

o    Note: If the profile does not exist you will proceed directly to the setup screen in Step 2.

Step 1a: Click NO. You do not want to overwrite the existing profile and you must create an additional profile.

 

Step 1b: Type in BestEdgeMail for the new profile name.

 

Step 2: Enter the primary email address and password for your mailbox.

Check the ‘Enable Remember Password?’ box to have this computer remember the password when Outlook launches the profile,

 

Congratulations! Once the email address and password are verified, your profile will be created and ready to use! 

Optional: Setting up a prompt to access your previous Outlook Profile/Mail: 

  1. In Windows, click Start > Control Panel > Mail.
    (If you don't see the Mail icon, in Windows XP you may need to click the Switch to Classic View link and in Windows 7 type Mail in the “Search Control Panel” box.)
  2. Click Show Profiles > Prompt for a profile to be used.
  3. You will be prompted each time to choose a profile.
